
Проєкт: Набір вправ на роботу з масивами та масивоподібними структурами в JavaScript

Опис
- Простий навчальний репозиторій з кількома завданнями, що демонструють роботу з масивами: мапінг, фільтрація, сортування та редукція.

Структура файлів
- [index.html](index.html) — завантажує модулі з папки `js/`.
- [js/task-1.js](js/task-1.js) — отримання імен користувачів через `map()`.
- [js/task-2.js](js/task-2.js) — фільтрація користувачів за наявністю друга (`filter()` + `includes`).
- [js/task-3.js](js/task-3.js) — сортування користувачів за кількістю друзів (використано `toSorted`).
- [js/task-4.js](js/task-4.js) — підрахунок загального балансу по статі (`filter()` + `reduce()`).

Запуск
- Відкрийте файл [index.html](index.html) у браузері (подвійний клік або через Live Server).
- Скрипти підключені як ES-модулі, тому найкраще запускати через локальний сервер для повної підтримки модулів.

Примітки
- Код демонстраційний — призначений для навчання та локальної перевірки в консолі браузера.
